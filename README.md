# RubyOnRails

## Ma première expérience avec Ruby On Rails

###### ETAPE 1 : Réalisation d'un CRUD basique

Génération de l'architecture du projet avec :

``` rails new "app name" ```

Pour lancer notre serveur nous utiliserons :

``` bin/rails server ```

La page par défaut sera alors :

``` http://localhost:3000/ ```
